# v-model

官网open in new window

Vue3 用 v-model代替了Vue2里的 v-model 和 .sync

    1. 在<input>里面的使用
<input v-model="xxx" />
1

上面代码等价于

<input :value="xxx" @input="xxx = $event.target.value" />
1

v-bind:value的数据绑定 和 @input的事件监听

    1. 在组件里面的使用
<CustomInput v-model="xxx" />
1

上面代码等价于

<CustomInput :modelValue="xxx" @update:modelValue="xxx = $event" />
1

如果是input组件, 为了使组件能像这样工作，内部的 <input> 组件必须：

  • 绑定 value attribute 到 modelValue prop
  • 输入新的值时在 input 元素上触发 update:modelValue 事件

这里是相应的代码：

<!-- CustomInput.vue -->
<script>
export default {
  props: ["modelValue"],
  emits: ["update:modelValue"],
};
</script>

<template>
  <input
    :value="modelValue"
    @input="$emit('update:modelValue', $event.target.value)"
  />
</template>

Xinput.vue组件

<template>
  <label>
    <div>{{ title }}</div>
    <input :type="password ? 'password' : 'text'" v-model="content" />
    <!--props属性 modelValue不能更改 , 所以声明一个data属性, 赋值了modelValue-->
    input组件的content : {{ content }} || props的modelValue : {{ modelValue }}
  </label>
</template>
<script>
export default {
  props: {
    password: {
      type: Boolean,
      default: false,
    },
    title: {
      type: String,
    },
    modelValue: {
      //以 外面的v-model值 以 modelValue 属性值传到组件里面
      type: String,
    },
  },
  data() {
    return {
      content: this.modelValue, //modelValue是props属性 , 所以不能更改, 所以用data属性代替
    };
  },
  watch: {
    content(newValue) {
      //监听data值 content , 每当他变化时通知外面更改
      this.$emit("update:modelValue", newValue);
    },
  },
};
</script>

  • 再添加XTextarea文本域组件

父组件里使用时

<XTextarea :rows="4" v-model="description" title="介绍"></XTextarea>
1

XTextarea.vue:

<template>
  <label>
    <div>{{ title }}</div>
    <textarea
      v-model="content"
      :rows="rows"
      @input="$emit('update:modelValue', $event.target.value)"
    ></textarea>
    <!-- @input="$emit('update:modelValue' , $event.target.value)" 输入时通知外面更改-->
  </label>
</template>
<script>
export default {
  props: {
    title: {
      type: String,
      required: true,
    },
    modelValue: {
      type: String,
    },
    rows: {
      type: Number,
      default: 3,
    },
  },
  data() {
    return {
      content: this.modelValue,
    };
  },
};
</script>
