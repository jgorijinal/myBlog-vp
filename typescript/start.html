# TypeScript 的运行环境

  • 方式一：通过 webpack，配置本地的 TypeScript 编译环境和开启一个本地服务，可以直接运行在浏览器上
  • 方式二：通过 ts-node库，为 TypeScript 的运行提供执行环境

# webpack 配置

目录结构

现在来开始配置 webpack

创建 package.json 文件

npm init
1

安装局部的webpack

npm install webpack webpack-cli -D
1

package.json script脚本

根目录下创建 webpack.config.js 配置文件

局部安装 ts-loader 和 typescript

npm install ts-loader typescript -D
1

在配置一下 webpack.config.js 执行 npm run build , 报错缺少 tsconfig.json 文件 执行命令 tsc --init 生成 tsconfig.json 文件

加上 extensions

在用 webpack-dev-server 搭建本地服务

npm install webpack-dev-server -D
1

想要告诉他默认的模板是根目录下的index.html, 需要使用 html-webpack-plugin 插件 console.log(message); 安装 html-webpack-plugin 插件到根目录里

npm install html-webpack-plugin -D
1

配置html-webpack-plugin 插件

然后最终执行 npm run serve

# 使用 ts-node

  • 方式二：安装 ts-node
npm install ts-node -g
1
  • 另外ts-node需要依赖 tslib 和 @types/node 两个包
npm install tslib @types/node -g
1
  • 现在，可以直接通过 ts-node 来运行TypeScript的代码
ts-node math.ts
1
